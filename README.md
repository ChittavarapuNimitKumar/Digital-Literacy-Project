🚀 Project Overview

This repository contains my Digital Literacy Portfolio, created as part of my assignment as a Student Digital Ambassador. The goal of this project is to build resources and establish a foundational digital presence to help peers navigate the digital world effectively, professionally, and safely.

This portfolio covers five core modules ranging from visual communication and professional profiling to coding platform familiarity, digital etiquette, and cybersecurity awareness.

Module 1: Digital Literacy Awareness Infographic

As a Digital Ambassador, I designed a one-page visual resource to help batchmates understand the importance of digital literacy, useful tools, and safe internet practices.

Tools Used: Canva

Module 2: Student Digital Portfolio

A professional digital presence is crucial for modern students. In this module, I set up my foundational professional portfolio across key academic and developer platforms.
